#+title: Gospel tuple lens
When we want to claim ownership of a tuple in [[id:27926a9e-056a-43bb-886e-37f4a973bf98][Gospel]], we can use the ~*~
operator in our lenses.  For example:
#+ATTR_LATEX: :environment ocamlenv
#+begin_example ocaml
val f : int array * int array -> unit
(*@ consumes arrs @ int array * int * array
    let _ = f arrs *)
#+end_example
